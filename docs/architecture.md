## Wolfcoin Architecture

### Phase One

Port "BasicCoin" from Python to Common Lisp

See: https://github.com/zack-bitcoin/basiccoin

#### Modules:

* ECDSA Cryptography
* P2P Protocol
* Transactions
* Blockchain
* Database
* HD Wallet
* REST API
* Web-based Client

#### Dependencies:

* Ironclad, Nibbles
* CL-STORE
* CL-Async, Wookie
* CL-WHO, Parenscript
* Trivial-Types
* Flexi-Streams
