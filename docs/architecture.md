## Wolfcoin Architecture

### Phase One

Port "BasicCoin" from Python to Common Lisp

See: https://github.com/zack-bitcoin/basiccoin

#### Modules:

* ECDSA Cryptography
* P2P Protocol
* Transactions
* Blockchain
* Database
* HD Wallet
* Miner
* REST API
* Web-based Client

#### Dependencies:

* Nibbles
* Ironclad
* CL-ISAAC
* BIT-SMASHER
* CL-STORE
* CL-ASYNC
* Wookie
* CL-WHO
* CL-CSS
* Parenscript
* Trivial-Types
* Flexi-Streams
